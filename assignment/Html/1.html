1). Define the terms: Website, Webpage, Web browser, Web server,HTML,CSS. -> Website: A website is a collection of related web pages typically served from a single web domain. It is designed to be viewed in a web browser and accessed via the Internet or a local network. -> Webpage: A webpage is a single document or resource of content that is accessible through the World Wide Web. It can contain text, images, videos, hyperlinks, and other multimedia elements. Webpages are designed using languages like HTML and styled using CSS. -> Web browser: A web browser is a software application used to access information on the World Wide Web. It interprets HTML documents and displays them as webpages, allowing users to navigate between different sites and interact with various online content. -> Web server: A web server is a computer program or hardware device that serves content to users over the Internet. It receives requests from web browsers and delivers webpages, files, or other resources in response. Common web server software includes Apache, Nginx, and Microsoft Internet Information Services (IIS). -> HTML (Hypertext Markup Language): HTML is the standard markup language used to create and design webpages. It provides the structure and layout of a webpage by using a system of tags and attributes to define elements such as headings, paragraphs, images, links, and forms. -> CSS (Cascading Style Sheets): CSS is a style sheet language used to control the presentation and appearance of HTML documents. It allows web developers to specify the colors, fonts, spacing, layout, and other visual aspects of a webpage, ensuring consistent design across multiple pages or entire websites. 2).CSS can be integrated into a web page in several ways, depending on the needs and preferences of the developer.
